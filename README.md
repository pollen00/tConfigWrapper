# tConfig Wrapper
tConfig Wrapper is a mod that allows for tConfig mods to be run within tModLoader.

### Progress
```diff
# Done:
+ Prefixes

# Doing:
! NPCs
! Items
! Tiles
! Projectiles
! Walls
! Dll loading
! Script loading

# Haven't started:
- Buff loading
- Sound loading
- Gore loading
- Global Classes
- Generic Classes
```
